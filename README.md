# Personal Branding Platform

## Start running in local env

```bash
sail up -d
sail artisan migrate
sail artisan key:generate
```

For Tailwind:

```bash
sail npm run dev
```


## General Features

- Auth
    - Register
    - Login 
    - Forget password
    - Reset password
    - Edit profile
